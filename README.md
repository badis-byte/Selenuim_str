Selenium Search & Click Script

A small Selenium automation script that opens a browser, searches for a term, and clicks a specific link from the results.

This project was made as a quick experiment / learning script for browser automation using Selenium.

Features

Launches a browser using Selenium

Searches for a given query

Finds and clicks a target link

Simple and easy to modify

Requirements

Python 3.x

Google Chrome (or another supported browser)
